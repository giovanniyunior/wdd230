Color Schema

  • Primary Background: #F7FFF6 (light, creamy white) for a clean and fresh feel.
  • Accents and Highlights: #BCEBCB (soft green) to add a calming and refreshing touch.
  • Text and Typography: #1E152A (dark navy) for legible and contrasting text against the light background.
  • Calls to Action: #931621 (deep red) for attention-grabbing buttons or interactive elements.
  • Secondary Background and Elements: #746F72 (medium gray) for subtle contrast and visual depth.
    •

    Typography

  • Heading Font: Ysabeau
    • The Ysabeau font will serve as the primary font for headings, including section titles and main headings.
    • The font weight can be adjusted to bold or semibold to add impact and emphasis to the headings.
    • Font sizes should be set based on the hierarchy of the headings, ensuring clear visual hierarchy.
  • Body Text Font: Open Sans
    • Open Sans, a clean and versatile sans-serif font, can be paired with Ysabeau for the body text.
    • Open Sans offers excellent readability and works well in various sizes and weights.
    • The font should be set to a regular weight for the main content paragraphs.
